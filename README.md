# linux-0.12

## Build on Linux
    $ cd src
    $ make help		    //get help
    $ make all 		    //compile & build
    $ make start	    //boot it on qemu with hard disk image
    $ make debug        //debug kernel via qemu and gdb
    $ make bochs-debug  //debug kernel via bochs and its own debugger
    $ make clean        //clean
